# docs/

Documentación viva de los productos en este repositorio (Field App y Sales Dashboard).

## Para qué sirve esta carpeta

- **Material de referencia** para futuras sesiones de Claude Code y para colaboradores humanos que necesiten contexto más allá de lo que está en `CLAUDE.md` o en el código.
- Cubre cosas que no se infieren leyendo el código: queries SAP canónicas, decisiones de arquitectura, estado de integraciones externas, checklists operativos, etc.

## Origen y vigencia

Los archivos aquí son **snapshots** tomados del workspace de Claude Project del proyecto. Esa es la fuente canónica — los documentos viven y se editan allá primero.

> ⚠️ Es posible que las versiones en `docs/` estén desactualizadas respecto al workspace de Claude Project. Si un documento parece contradecir información reciente, verifica contra la versión del workspace antes de actuar sobre él, y actualiza el snapshot aquí cuando sea posible.
